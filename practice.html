Building a Web Page

HTML, known as Hyper Text Markup Language, is the script text that allows us to create a web page content.

Hyper Text Markup Language Tags

HTML consists on markup tags. Most have an opening and a closing tag. However, some like the image tag do not have a closing tag.

These are some of the most common tags:

  • paragraph - p
  • header 1 - h1
  • header 2 - h2
  • list item - li
  • unordered list - ul
  • ordered list - ol
  • anchor - a
  • image - img

Cascading Style Sheet

The Cascading Style Sheet defines the look and style of a web page . It helps define the:

  1. color
  2. Font size
  3. Font style
  4. Position of the content on the page
